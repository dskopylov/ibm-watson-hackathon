Math Processing Task
В данном тесте Вам необходимо определить будет ли являться результат показанного на экране математического выражения меньше пяти, или же он будет больше пяти. Ответ необходимо указать с помощью нажатий на клавиатуре клавиш "S" или "J" соответственно. На каждый блок заданий, отличающихся между собой количеством слагаемых, отводится по 3 минуты. Следующее задание появляется после ответа на текущее задание или по истечении времени на него. Для начала выполнения теста нажмите кнопку.
►
В текущем блоке выражения будут содержать по числа. Ограничение по времени для каждого выражения – секунды. Желаем удачи! Для начала выполнения задания нажмите клавишу "Пробел".
Меньше пяти (S)
Больше пяти (J)
